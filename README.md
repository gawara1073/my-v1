# コマ合わせ 📅 | 大学生向け空き時間チェッカー v2

LINEでの面倒な予定調整を、タップだけで一瞬で解決する大学生向け予定調整ツール「コマ合わせ」のアップグレード版 v2 です。
サーバーやデータベースは不要。HTML/CSS/JavaScriptのみで動作し、**GitHub Pages** にデプロイするだけで即座に無料で公開・共有可能です。

---

## 🌟 v2でのパワーアップ要素（主な機能）

1. **その場でのメンバー個別編集・更新機能**
   - 登録したメンバーの横にある「編集（ペンシルアイコン）」ボタンから、登録済みのスケジュールや名前を時間割グリッド上で直接修正・上書き保存できます。
2. **メンバー別の自動カラー割り当て＆色分け表示**
   - メンバーが登録されると、自動的に固有のテーマカラー（最大10色）が割り当てられます。
   - **ドットインジケーター**: ヒートマップの各セル内に、空いているメンバーのカラーを示す小さなドットが並び、ホバーしなくても誰が空いているのか一目で分かります。
   - **詳細表示パネル**: セルをタップすると、そのコマで空いているメンバーがカラーバッジ付きで下部パネルに一覧表示されます。
3. **全員空いている共通コマ（パーフェクトマッチ）の特別ハイライト**
   - メンバー全員が空いているコマ（一致率100%）がある場合、グリッド上でゴールドに光り輝く特別な装飾と波紋アニメーションで強力に目立たせます。
4. **スマホに特化した「曜日タブ」＆シングルデイ・ビュー**
   - スマートフォンなど幅の狭い画面では、上部の「曜日タブ」で曜日を切り替え、その日のコマだけを大きく縦に表示します。誤タップを防ぎ、片手でも快適に選択が可能です。
5. **日程決定＆LINE用テキスト自動生成**
   - おすすめ日程の「この日程に決定」ボタンを押すと、LINEグループにそのまま貼り付けて送信できる「決定通知テキスト」が自動生成され、ワンタップでコピーできます。
6. **アニメーション演出とプレミアムUI/UX**
   - メンバーの追加・更新が完了した際や、日程決定時に画面全体に「紙吹雪（Confetti）」が舞う楽しいインタラクションを搭載。
   - ガラスモルフィズム風の洗練されたモダンUI、ダークモードへの完全対応。

## 🚀 LINEでの予定調整の流れ

1. **主催者が予定を作る**
   - 「コマ合わせ」のウェブサイトを開き、自分の名前（例: `たろう`）を入力します。
   - 時間割グリッドから空いているコマをタップ（またはなぞり選択）し、「メンバーに追加する」を押します。
   - 「LINEに共有URLをコピー」ボタンを押し、コピーされたURLをLINEグループに送信します。

2. **友達が予定を重ねる**
   - 友達がLINEから送られたURLを開くと、**すでに「たろう」さんの空き時間が反映された状態**で起動します。
   - 友達も自分の名前（例: `はなこ`）を入力して、空いているコマを選び「メンバーに追加する」を押します。
   - 再度「LINEに共有URLをコピー」を押し、最新のURLをLINEに送ります。これを繰り返します。

3. **最適な日程を決定し、LINEで通知する**
   - メンバーが登録されると、全員が空いている「超おすすめ」コマや、最も集まりやすい日程が「💡 おすすめの時間帯」に自動で提案されます。
   - 決定したい日程の「この日程に決定」ボタンを押すと、LINE送信用テキストが生成されるので「コピー」してLINEに貼り付ければ、一瞬で調整が完了します！

## 🌐 GitHub Pages への公開手順

このプロジェクトは静的ホスティング（GitHub Pages、Vercelなど）で今すぐ無料で世界中に公開できます。

1. **GitHubリポジトリを作成する**
   - GitHubにログインし、新しいリポジトリ（例: `koma-awase`）を作成します。

2. **コードをプッシュする**
   - ローカルのプロジェクトディレクトリにて以下のコマンドを実行し、GitHubにプッシュします。
   ```bash
   git init
   git add .
   git commit -m "feat: upgrade to v2"
   git branch -M main
   git remote add origin <あなたのGitHubリポジトリURL>
   git push -u origin main
   ```

3. **GitHub Pages を有効化する**
   - リポジトリの **[Settings]** タブ ＞ **[Pages]** を選択します。
   - **Build and deployment** の **Branch** で `main` を選択し、フォルダを `/ (root)` にして **[Save]** をクリックします。
   - 数分後、`https://<あなたのユーザー名>.github.io/koma-awase/` にて公開されます！

## 🎨 技術的な解説（フロントエンド完結）

- **URLシリアライズによるサーバーレス設計**:
  登録されたメンバー全員のスケジュールデータ配列をJSON文字列化し、URIエンコードとBase64を組み合わせて1つのURLパラメータ（`?d=...`）に圧縮・保持しています。データベースがないため、読み込みが高速かつ個人情報もサーバーに送信されない安心設計です。
- **マルチバイト文字（日本語名）の文字化け対策**:
  ブラウザの `btoa()` / `atob()` はUnicode文字をそのまま変換するとエラーを起こすため、UTF-8バイト配列のエスケープシーケンスに変換してからシリアライズを行っています。
- **純粋なCSSとCanvasによる軽量アニメーション**:
  Confetti（紙吹雪）は外部ライブラリを一切使わず、HTML5の `<canvas>` 上にJavaScriptで直接パーティクルを描画・降下させており、表示負荷が極めて低く抑えられています。
